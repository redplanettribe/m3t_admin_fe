# M3T Admin

Event admin SPA built with Vite, React 19, TypeScript, Tailwind CSS, shadcn/ui, React Router, TanStack Query, and Zustand.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Stack

- **Build**: Vite  
- **Framework**: React 19 + TypeScript  
- **Styling**: Tailwind CSS  
- **Components**: shadcn/ui  
- **Server state**: TanStack Query  
- **Routing**: React Router  
- **Client state**: Zustand
